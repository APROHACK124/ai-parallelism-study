# Phase 2

The objective of the phase 2 is to implement various pytorch modules using only `torch`, `nn.Module`, `nn.Parameter`, `torch.autograd`, `torch.nn.init`, as well as basic tensor operations.

This involves the creation of a small library of such modules for transformations, loss functions, and the AdamW optimizer.

At the end, the idea is to implement a functional transformer block pre-norm, tests to compare with pytorch.
